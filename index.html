Synchronisation cloud

Sauvegarde en ligne gratuite : tes paris survivent aux mises à jour et même à la perte du téléphone. Une fois connecté, sauvegarde automatique toutes les 6 h.

Guide d'installation (5 min)
  1. Crée un compte gratuit sur supabase.com, puis un nouveau projet.
  2. Dans SQL Editor, colle et exécute : 
    create table bk_sync (
  id text primary key,
  data jsonb,
  updated_at timestamptz
);
alter table bk_sync enable row level security;
create policy "bk_all" on bk_sync
  for all using (true) with check (true);
  3. Dans Settings → API : copie l'URL du projet et la clé anon public dans les champs ci-dessous.
  4. Invente un code de synchro long et unique : c'est lui qui protège tes données.
